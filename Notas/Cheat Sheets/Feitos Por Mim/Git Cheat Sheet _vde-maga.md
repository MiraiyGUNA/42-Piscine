# Git Cheat Sheet _vde-maga

## Estado dos Ficheiros

```bash
git status
```

## Adicionar Ficheiros ao Repo

```bash
git add hi.txt
```

## Aviso de Mudanças antes do Commit

> -m adiciona uma mensagem


```bash
git commit -m "added hi.txt"
```

## Dar Upload para o Repositório 

```bash
git push
```
